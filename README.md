## 👋 我是彭淑文（Apple Peng）

我常笑著說，我的工作其實就是「中翻中」。

把使用者說不清楚的需求，翻譯給工程師。

再把工程師的技術語言，翻譯成主管與使用者都能理解的內容。

因工作與專案需求，我經常需要快速理解陌生領域。

每當接觸新的領域或任務，我習慣先理解整體架構，再開始分析與整理。
