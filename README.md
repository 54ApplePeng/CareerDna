# Apple Career DNA

## 👋 我是彭淑文（Apple Peng）

因工作的轉換或是專案需要經常接觸陌生領域。

每到一個新產業，我總是先理解它的語言、流程與規則，再把需求整理成工程師、主管、使用者都能理解的內容。

二十多年來，我跨足政府、教育、醫療、半導體、物流、專利、數位內容等不同領域，角色也從工程師一路做到系統分析師、PM、資料分析、內部講師。

我最擅長的，不是某一項技術，而是把複雜的事情整理清楚，讓不同角色的人可以一起完成事情。
