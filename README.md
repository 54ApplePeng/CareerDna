# Apple Career DNA

## 👋 我是彭淑文（Apple Peng）

我常笑著說，我的工作其實就是 **「中翻中」**。

把使用者說不清楚的需求，翻譯給工程師；再把工程師的技術語言，翻譯成主管與使用者都能理解的內容。

因工作轉換或專案需求，我經常需要快速接觸陌生領域。

一路走來，曾參與政府、教育、醫療、物流、專利、數位內容等不同領域的資訊專案，也曾擔任工程師、系統分析師、PM、資料分析、內部講師等不同角色。

我喜歡理解新的領域，也喜歡把複雜的事情整理清楚，讓不同角色的人可以一起完成事情。

---

## 🧬 我的 Career DNA

- 🧩 快速理解陌生領域
- 🤝 串聯需求、流程與系統
- 🗣 擔任不同角色之間的「中翻中」
- 📊 用資料協助分析與決策
- 🤖 持續探索 AI 與流程自動化的可能性
